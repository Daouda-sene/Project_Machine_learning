## Machine Learning Project

### Project Description

This project demonstrates the implementation of **Linear Regression using Gradient Descent** to learn the relationship between input variables and target values.

The objective of the project is to understand how machine learning models learn from data by minimizing a cost function through iterative optimization.

The model predicts the output ( y ) using the linear hypothesis:

[
f(x) = wx + b
]

Where:

* **w** represents the weight (slope)
* **b** represents the bias (intercept)

The parameters are optimized using **Gradient Descent**, which iteratively updates the parameters to minimize the cost function.

### Key Features

* Implementation of the **cost function**
* Computation of **gradients**
* Implementation of **gradient descent optimization**
* Visualization of the training process
* Prediction of outputs for new data
